## üîí Security & Best Practices

### **Wallet Management**
- ‚úÖ **Auto-Generated Wallets** - Project wallets are created automatically by the bot
- ‚úÖ **Encrypted Storage** - All sensitive data is encrypted before database storage
- ‚úÖ **New Wallets Only** - Always use dedicated wallets (never personal or exchange wallets)
- ‚úÖ **Minimal Storage** - Keep only necessary tokens in bot wallets (top up regularly)

### **Wallet Requirements**
Each project wallet needs:
- **EGLD** - For blockchain transaction fees
- **REWARD tokens** - For MakeX API usage fees ($0.03 per on-chain transfer)
- **Supported tokens** - The tokens you want to send/tip

### **Usage Fees**
- **Virtual Account Operations** - FREE (tips, games, NFT transfers within virtual accounts)
- **On-Chain Transfers** - $0.03 in REWARD tokens (only for admin transfers and withdrawals)

{% hint style="warning" %}
**Important Security Note:** Always create new, dedicated wallets for the bot. While PEM files are encrypted in the database, it's best practice to keep minimal funds in bot wallets and top up regularly. Never use personal or exchange wallets.
{% endhint %}
